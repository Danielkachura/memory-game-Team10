# UI Module

Shared UI building blocks for Squad RPS.
- Button components
- Panel components
- Layout wrappers


## Owns
- reusable buttons and badges
- modal and overlay primitives
- shared visual helpers such as star rating
- design-system-level styling primitives

## Planned Structure
- `src/components/` for reusable UI components
- `src/styles/` for shared UI-layer styling helpers
