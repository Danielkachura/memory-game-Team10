# ai Module

Claude-powered Memory Game features.

## Owns
- prompt builders
- frontend Claude client wrapper
- theme generation flow
- hint flow
- end-game narration flow

## Planned Structure
- `src/prompts/` for theme, hint, and narrator prompts
- `src/services/` for client wrappers and AI feature services
- `tests/unit/` for prompt and client logic
- `tests/integration/` for failure and fallback behavior
